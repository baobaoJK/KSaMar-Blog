# 简介

本文介绍了如何在 Unity 中使用 VS Code 进行游戏开发

# 下载必要的开发环境安装包

进行 Unity 开发必不可少的就是 .NET (C#) 与 .NET Framework
本文使用的 Unity 版本是 2022.3.17f1c1
.NET 的版本是 8.0.406
.NET FrameWork 的版本是 4.8.1

.NET 8.0 下载链接：点我跳转

点击 .NET 8.0 下的 Download .NET SDK x64 即可

.NET Framework 4.8.1 下载链接：点我跳转

点击 Download .NET Framework 4.8.1 Developer Pack 即可

# 在 Unity 中设置 External Script Editor

重要：
你可能会遇到当 Unity 重启之后，原来设置的 External Script Editor 又恢复默认状态而不是 VSCode

解决办法：
使用管理员身份运行 Unity Hub 后再打开 Unity 进行设置

当你选择用管理员身份运行时，打开 Unity Editor 时请选择 I wish to continue at my own risk 打开 Unity Editor

在 Unity 中找到 Edit -> Preferences… -> External Tools，然后点击 External Script Editor 并选择 VSCode 快捷键方式或者 EXE 程序

# 检查 Unity 中是否安装了 Visual Studio Code Editor 资源包

首先打开 Window -> Package Manager

在 Package Manager 中选择 Packages: Unity Registry 并搜索 vs 差看是否已安装，已安装会显示 Remove 按钮

# 前言

名称：Fast-Chat-X / 极聊 X

作者：KSaMar

日期：2024/08/25 - 2024/08/27

描述：一个在线的即时聊天网页聊天室

# 简介

采用 Vue3 + Python3 + Flask + SocketIO 的即时在线多人聊天室网页项目（客户端）

功能模块：用户登录、用户注册、用户忘记密码、群聊等（未来会弄更多）

可以发送文本 / 符号表情与 emoji 表情 😊

消息可以存取到数据库中

# 开发环境

开发软件：VS Code / PyCharm 2023.2.3

Node 版本：18.16.0

Python 版本：Python 3.7

# 技术栈

项目采用以下技术：

前端：Vue3 + Scss + TypeScript + Element-Plus + Axios + SocketIO

后端：Python3 + Flask + SocketIO + SQLAlchemy + MySQL
